HTML

  • I learned more about the head elements such as meta and title
  • Meta has lots of attributes including charset and viewport
  • A child element rests within another, while a parent element contains one or more
  • Elements encompass the entire element; tags include only that which is within the angle brackets

CSS

  • CSS stands for Cascading Style Sheets and its purpose is to make HTML look pretty
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are 3 types of CSS: inline, internal style sheet and external style sheet. The last one is preferred
  • To style all of an element type, use the name of the element
  • .name indcates a class, which styles all of the elements with that class name

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git config: sets up configuration options such as name and email
  • git commit: creates a snapshot of the current build
  • git add -A: commits all files to the working branch
  • git pull origin main: checks that the main branch is up to date
